SumBroo

Guest-Posting For Social Media Creators

Web Dev

SumBroo

SumBroo was a project I poured my heart into as a potential startup, aiming to solve a significant issue for social media creators. The platform was designed to allow creators to automatically guest post on each other's accounts, making collaboration easier and more efficient. Despite the effort, the project did not succeed as I couldn't get social media creators to see the value in the platform.

Key Aspects

Design System: The design was carefully crafted to appeal to social media creators, with a focus on a modern, user-friendly interface that would make the process of collaboration as seamless as possible. The platform’s aesthetics were intended to resonate with a creative audience, while the UX design prioritized ease of use.

Code Structure: From a technical standpoint, the code structure was built with scalability and performance in mind, using a robust stack that could support the platform’s features: MongoDB as a database, NextJS, HTML, CSS, and JS for the front-end, and ExpressJS & NodeJS for the backend. The platform was designed to be responsive and efficient, ensuring a smooth experience for users across various devices and social media platforms.

Platform Copy: The copy on the site was informal and relatable, aimed at speaking directly to social media creators. It conveyed the challenges they face in collaboration and how SumBroo was designed to solve those problems. The about section, in particular, was personal and heartfelt, sharing my own journey and the inspiration behind the platform, in the hopes of connecting with creators on a deeper level.
